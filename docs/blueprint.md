# **App Name**: Hariharan Portfolio

## Core Features:

- Dynamic Home Display: Display the 'Home' section, featuring a split layout with personal content on the left and a profile image on the right, and a typing animation for listed roles.
- Interactive About Section: Render the 'About' section, incorporating a hero image, a rich text description of skills and interests, and an interactive skills showcase using icons.
- Education Timeline: Generate a timeline layout for the 'Education' section, using educational cards with institution logos and a progress indicator for current studies.
- Projects Grid: Implement a card-based grid layout for the 'Projects' section, enabling filtering by technology or category and including search functionality. Display project details in a modal popup.
- Blog Showcase: Showcase a card layout for the 'Blogs' section, complete with featured images and rich HTML content rendering. Add search, filtering by tags, and estimated reading time.
- AI-Powered Chatbot Assistant: Integrate an AI-powered chatbot that serves as a tool to guide visitors through the portfolio by understanding their queries and directing them to relevant sections or information.
- Admin Authentication: Protected admin route for login/logout.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5), evoking trust and stability suitable for a professional portfolio.
- Background color: Light Gray (#ECEFF1), a desaturated version of the primary, for a clean, unobtrusive backdrop.
- Accent color: Yellow-Amber (#FFC107), analogous to blue with a distinct contrast in brightness and saturation, highlights important interactive elements.
- Body text: 'PT Sans' (sans-serif) offers a modern yet warm and accessible reading experience for descriptions and general content; Headlines: 'Playfair' (serif), bring an elegant, high-end touch.
- Use minimalist icons from Lucide React to maintain a clean and modern aesthetic. Ensure icons are consistent in style and weight across the portfolio.
- Employ a responsive, grid-based layout adapting to various screen sizes. Use clear, well-defined sections for each content category to facilitate easy navigation and readability.
- Incorporate subtle animations using Framer Motion for page transitions, hover effects, and content reveals to add interactivity without overwhelming the user.
